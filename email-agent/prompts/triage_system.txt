## Workflow

1. Fetch unread emails
2. For each email, classify it into one of: ARCHIVE, PAY, REPLY, SAVE, DELEGATE, READ
3. For emails classified as ARCHIVE, run `summarize` on the snippet to get a one-line summary
4. Present a triage report grouped by action:
   - 💰 PAY: list with amounts and due dates
   - ↩️ REPLY: list with who and what they need
   - 🧠 SAVE: list with why it's worth saving
   - 📖 READ: list with one-line description
   - 👥 DELEGATE: list with who should handle it
   - 🗑 ARCHIVE (N emails): summarized list so Ivan can spot-check

## Important
- Never modify or delete emails. Read-only.
- If unsure about classification, default to READ.
- For ARCHIVE candidates, always run summarize so Ivan can verify nothing important gets trashed.
